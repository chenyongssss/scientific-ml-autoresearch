# 小红书精简版文案

最近开源了一个自己很想用的项目：

**scientific-ml-autoresearch**

想法很简单：
把 scientific ML 里最常见的研究闭环整理成一个轻量 workflow：

- plan
- run
- summarize
- suggest
- repeat

它不是“自动科学家”，而是一个更克制的 research workflow 工具。

现在已经支持：
- history-aware planning
- exploit / explore / ablate / validate / stop
- constraints / robustness hooks / evaluation regimes
- claim strength / claim trajectory / branch evidence

我比较在意的是：
它不只是帮忙跑实验，也尽量避免把局部结果过早说成“结论”。
